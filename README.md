# 🕹️ 2D Street Fighter-Style Unity Game

A 2D **Street Fighter-style** game built in Unity featuring:
- ⚔️ **Player vs Player** and **Player vs Computer** modes  
- ❤️ Health bars, attack effects, and fluid character animations  
- 🎮 Keyboard controls for movement and attacks 
- 🖥️ Clean and functional UI system

---

## 🕹️ Controls

### 🧭 Adventure Mode (Single Player)
- `A` – Move Left  
- `W` – Jump  
- `D` – Move Right  
- `E` – Attack  
- `Q` – Heavy Attack (available in **Levels 3 & 4**)

---

## 🎮 How to Play

1. Clone or download the project
2. Open it in **Unity**
3. Press **Play** to start testing
4. Choose **Adventure** or **1v1** mode from the main menu
5. Fight, win, and survive the hardest levels!
